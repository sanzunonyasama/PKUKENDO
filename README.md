
## 介绍

这是一个托管在[AVOS Cloud](https://cn.avoscloud.com)上的一个简单留言板网站。

访问[http://myapp.avosapps.com](http://myapp.avosapps.com)即可访问在线演示版本。

因为这只是一个演示项目，因此源码公开，可以直接看到App Id和App Key。实际项目请妥善保管您的源码和App信息。尽量使用Git私有仓库和本地部署功能。
